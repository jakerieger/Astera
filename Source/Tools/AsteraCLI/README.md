# AsteraCLI

AsteraCLI (or `astera-cli` when called from the command line) is a project management and engine
helper tool. It allows you to create and manage projects, generate asset descriptors, and *build
scene files from XML.

> * work in progress

```
Usage: ./astera-cli [OPTIONS] SUBCOMMAND

Options:
  -h,--help                   Print this help message and exit

Subcommands:
  project                     Manage projects
  asset                       Manage assets
```
